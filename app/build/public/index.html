JSON Server

Endpoints:

  • /jsonplaceholder/api/albums
  • /freecodecamp/authors
  • /jsonplaceholder/api/comments
  • /jsonplaceholder/api/posts
  • /jsonplaceholder/api/todos
  • /jsonplaceholder/api/users
To return from endpoint, right click and select "back"

Use with Fetch or Axios

Fetch

//function goes in place of console.log fetch("http://localhost:8181/freecodecamp/authors/") .then(res => res.json()) .then(res => console.log(res)) const getUser = async() => await fetch("http://localhost:8181/jsonplaceholder/api/users") .then(res => res.json()) .then(res => console.log(res)) getUser()

Axios

//function goes in place of console.log const posts = axios.get("http://localhost:8181/jsonplaceholder/api/posts") .then(res => console.log(res.data)) const getTodos = async () => await axios.get("http://localhost:8181/jsonplaceholder/api/todos") .then(res => console.log(res.data)) getTodos()

Datasets

The authors dataset is from freecodecamp.org. All the resources have been scraped from the freecodecamp site and are served up on the local host.

Freecode camp is an excellent free, open source resource, and the base of our curriculum

The jsonplaceholder endpoints are from jsonplaceholder.com, a great free open source resource on the Web
